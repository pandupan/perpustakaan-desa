# 📚 Perpustakaan Desa

Sistem **manajemen perpustakaan desa** berbasis web. Aplikasi untuk mengelola koleksi buku, peminjaman, dan pendataan anggota perpustakaan di tingkat desa.

## ✨ Fitur

- **Manajemen Buku** — Tambah, edit, hapus data buku
- **Peminjaman** — Proses peminjaman dan pengembalian buku
- **Anggota** — Data anggota perpustakaan
- **Laporan** — Rekap data peminjaman

## 🛠️ Tech Stack

- **Next.js** — React framework
- **TypeScript** — Type safety
- **Tailwind CSS** — Styling
- **shadcn/ui** — UI components

## 🚀 Cara Menjalankan

```bash
npm install
npm run dev
```
